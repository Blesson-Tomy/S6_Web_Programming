FAQ - Introduction to Computer Network

Questions

What does TCP stand for?

What is the main function of TCP in networking?

What does IP stand for?

What is the primary purpose of the IP protocol?

What does ICMP stand for?

What is the main use of ICMP?

How does TCP ensure reliable data transmission?

What is the difference between TCP and UDP?

What is an IP address?

Give an example of an ICMP message type.

Answers

Answer 1: TCP

Transmission Control Protocol.

Answer 2: What is TCP?

TCP provides reliable, connection-oriented data transmission between devices.

Answer 3: IP

Internet Protocol.

Answer 4: Why IP?

IP is responsible for addressing and routing packets between devices across networks.

Answer 5: ICMP

Internet Control Message Protocol.

Answer 6: Why ICMP?

ICMP is used for sending error messages and operational information, such as with the ping command.

Answer 7: What is TCP?

TCP uses acknowledgments, sequence numbers, and retransmissions to ensure data is delivered correctly.

Answer 8: Comparison

TCP is connection-oriented and reliable, while UDP is connectionless and does not guarantee delivery.

Answer 9: What is IP?

An IP address is a unique identifier assigned to each device on a network.

Answer 10: ICMP Function

Echo Request and Echo Reply messages, used by the ping utility.
